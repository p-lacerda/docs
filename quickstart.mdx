## Prepare seu ambiente de desenvolvimento

Antes de tudo, é importante garantir que você tenha as ferramentas corretas instaladas para trabalhar com a stack do PediuAí:

- .NET 8  
- Entity Framework Core  
- SQL Server  
- SignalR  
- WhatsApp Business API  
- Azure (para hospedagem)  
- Docker

### Etapas iniciais

<AccordionGroup>
  <Accordion icon="laptop-code" title="Clonar o projeto">
    O repositório principal do PediuAí está hospedado no GitLab:
    
    ```bash
    git clone https://gitlab.com/sua-org/api-pediuai.git
    cd api-pediuai
    ```

    Certifique-se de ter o [Git](https://git-scm.com/downloads) instalado.
  </Accordion>

  <Accordion icon="cogs" title="Rodar o backend com Docker">
    Para subir o ambiente local com o Docker (incluindo .NET, SQL Server e dependências):
    
    ```bash
    docker-compose up --build
    ```

    O `docker-compose.yml` já inclui suporte para banco, backend e SignalR.
  </Accordion>

  <Accordion icon="network-wired" title="Configurar variáveis de ambiente">
    Copie o arquivo de exemplo `.env.example` e edite suas configurações:

    ```bash
    cp .env.example .env
    ```

    Preencha as variáveis relacionadas ao banco de dados, Azure e WhatsApp Business API.
  </Accordion>

  <Accordion icon="database" title="Aplicar migrations do Entity Framework">
    Após o banco estar de pé, aplique as migrations:

    ```bash
    dotnet ef database update
    ```

    Instale o EF CLI, se necessário:

    ```bash
    dotnet tool install --global dotnet-ef
    ```
  </Accordion>
</AccordionGroup>

### Deploy em produção

<AccordionGroup>
  <Accordion icon="cloud" title="Publicar no Azure via GitLab CI/CD">
    Use o GitLab CI/CD para automatizar o deploy no Azure:

    - Configure variáveis no GitLab (`AZURE_WEBAPP_NAME`, `AZURE_CREDENTIALS`, etc).
    - No repositório `api-pediuai`, adicione um arquivo `.gitlab-ci.yml` com os steps para build e deploy.
    - Utilize `az login` via Service Principal no pipeline para autenticar com o Azure.
  </Accordion>

  <Accordion icon="whatsapp" title="Configurar WhatsApp Business API">
    Integre a API oficial do WhatsApp:

    - Tenha uma conta verificada no [Facebook Business Manager](https://business.facebook.com/).
    - Configure o Webhook no Meta.
    - Use o token no campo `WHATSAPP_API_TOKEN` do `.env`.

    Documentação oficial: [developers.facebook.com/docs/whatsapp](https://developers.facebook.com/docs/whatsapp)
  </Accordion>
</AccordionGroup>
